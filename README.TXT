Instructions on how to operate the application:
1. Select a date from the drop down menu.
2. Press submit.
3. The resulting NVDA stock prediction and trading strategy recommendation tables will be printed.

Commit history can be found on the Github: https://github.com/ericcardinal99/SmartTrader_App

Build Instructions for Running the App Locally:
1. Installing Libraries:
    pip install flask
    pip install numpy
    pip install pandas
    pip install datetime
    pip install scikit-learn
1. Go to the program directory and run this line "python front_end_mlp.py"
2. Go to on "http://127.0.0.1:5001" on a web browser or simply click the link in the terminal to run the application locally.
